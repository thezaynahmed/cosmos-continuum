# Cosmos Continuum

## Overview
Cosmos Continuum is a futuristic, Augmented Reality (AR) healthcare platform designed to reimagine patient experiences. The project focuses on "scrollytelling" to showcase high-end, immersive products that aid in healing and therapy.

## Core Products

### 1. StoryWall ColorDive
**Theme:** Playful, Jungle/Ocean.
**Description:** An immersive mural experience revealed through parallax scrolling. It utilizes glassmorphism cards to explain how the product transforms clinical environments into engaging stories.

### 2. Breathe With Me
**Theme:** Calm, Dark Mode with Neon Accents.
**Description:** A rhythmic, breathing assistance tool featuring glowing animations and a deep, soothing aesthetic to help patients regulate their breathing.

## Tech Stack
- **Framework:** Next.js
- **Styling:** Tailwind CSS (Custom configurations for colors, typography, and animations)
- **Language:** TypeScript
- **Design Philosophy:** Premium, "Site of the Day" aesthetic, high-quality micro-interactions, and smooth scroll-triggered animations.

## Getting Started
Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
